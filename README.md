ChileanRut
===========

Validates, formats and unformats chilean rut (chilean tax Number)

To install
```
	$ gem install chilean_rut
```

To use
```
  require 'chilean_rut'


  ChileanRut.validate('')
  ChileanRut.format('')
  ChileanRut.unformat('')
  ChileanRut.validValidatorDigit('')
  ChileanRut.correctValidatorDigit('')
  ChileanRut.getValidatorDigit('')
```
